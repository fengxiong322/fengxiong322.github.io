Basic GUI

The first thing to learn about GUI is how to actually create a window to display your information. In the Java swing library, this is called a JFrame. JFrames are made to display information on the screen. You may add components into a JFrame, which allows it to display data. To find out how to use JFrames and add simple things like buttons, vist JFrame introduction.

Functionality of GUI

Now, we can explore some of the things you can do with java gui. One of the things you will learn is including things like buttons and drawings into you JFrame. Another thing you will learn is ActionListeners. Your JFrame should implement ActionListener if you would like to have interaction between the user and the program. ActionListeners can be added to different components, and can allow you program to respond things like user clicks.

You can checkout the first gui lesson for more information
