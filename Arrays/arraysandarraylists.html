Arrays And ArrayLists

In this article, you will learn what arrays and ArrayLists are, and their practical applications.

Arrays

So what really is an array? An array is a container, that is capable of holding a group of things. Imagine you wanted to represent a highway full of cars. Sure, you can create variables for each Car object, but what happens when you have 10 cars? 100 cars? 1000 cars? Your code would quickly become messy. The solution? Arrays.

To see an array in action, lets simulate a very simple road. Lets start by creating our Car class. For our purposes, we will just define the color of the car

public class Car{
  private String color;

  public Car(String newColor){
    color = newColor;
  }

  public String getColor(){
    return color;
  }

  public boolean equals(Car anotherCar){//If two cars have the same color, they are considered to be the same car
    if(this.getColor() == anotherCar.getColor())
      return true;
  }
}

ArrayLists

The next topic will be arraylists. An arraylist is a convient resizable array. Normal arrays are stuck with one size, and to change it, you have to manually create a new array of a different size and copy the original contents.
