Welcome to OOP!

This section contains all of my past OOP work in class. But what exactly is OOP?

The acronym "OOP" stands for Object Oriented Programming. This means that it attempts to model the physical world. In this OOP tutorial, I will be using our everyday life as an example.

Basic OOP

In OOP, there are two things that make up an "object": Its fields, or characteristics, and its methods, or the things it can do. In this way, we can define complex objects, like a human. Each and every human has some properties that are the same, for example, hands, nose, arms, legs, eyes, etc. Each human also has methods, or actions that they can perform, for example, running, jumping, eating and sleeping.

More can be found about this topic in Fairy Tale OOP
