The Instance Variables

Imagine we have a class, that describe the state of a certain Animal. For instance, its name, if its tired or hungry, and its weight. These things are declared as instance variables. These are put at the top of the class, after the opening line of the class. Instance variables should be private. This is to prevent other classes from making changes that shouldn't be possible. For instance, something may try to multiply the weight of an Animal by 100. This may not make sense. We will talk more about how we can prevent this later.

public class Animal{
  //Instance variables go here
  private String name;
  private boolean tired;
  private double weight;

So far, our class defines fields. They define a section of memory, in which information is to be stored. In general, whenever we use this class, we don't want these fields to be empty. We want to define something in it. To do this, we create a constructor. This constructor will create the Animal. Every Animal is different, so we should give a way to them their own name, and their weight when we create them. We can allow for these changes through parameters. Parameters is information that is given when the constructor is needed

  //Constructor name MUST be the same as the class name

    tired = true; // The animal will always start tired
    hungry = true; // The animal will always start hungry
    weight = w;
  }

Next, we want our animal to be able to interact with everything around it. For instance, if we feed it food, we want it to not be hungry. Likewise, if it sleeps, we want it to not be tired. We can do this by using methods. Methods define an action to perform.

  //The first method feed.
  public void feed(String v) {//feeds the animal
    hungry = false;//sets the hungry variable to false, making the animal not hungry
      if (v.equals("vegetables"))//If you chose to feed the animal "vegetables"...
        weight += 2;//The animal's weight increases by 2
      else if (v.equals("meat"))//If you chose to feed the animal "meat"...
        weight += 4;//The animal's weight increases by 4
      else//If you chose to feed the animal anything else...
        weight += 1;//The animal's weight increases by 1
  }

  //Second method
  public void sleep() {//makes the animal sleep
    tired = false;//Sets the tired variable to false, making the animal not tired
    weight -= 3;//The animal's weight decreases by 3
  }

Finally, we want to be able to run your program. We can do this by declaring a main method. This method is where you use all of the other methods to create a meaningful result. In this example, we make a few different animals

  public static void main(String[] args) {
      Animal a = new Animal("Cow", 10);
      Animal b = new Animal("Sheep", 10);
      Animal c = new Animal("Hippo", 10);
      Animal d = new Animal("Cat", 10);
      Animal e = new Animal("Goose", 10);
  }
}
